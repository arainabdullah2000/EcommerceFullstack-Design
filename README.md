A simple ecommerce website clone project built for learning and practice.
Currently focused on the frontend (master branch), with backend development planned next.

Features (Frontend)

Responsive design using HTML, CSS, and JavaScript
Product listing with images, prices, and details
Shopping cart UI for adding/removing items
Basic checkout page layout

Project Structure

ecommerce-clone/
│── frontend/        # Master branch (UI only)
│   ├── index.html
│   ├── style.css
│   ├── script.js
│── backend/         # Coming soon (Node.js + MongoDB)
│── README.md

Tech Stack

HTML, CSS, JavaScript for frontend
Planned backend: Node.js, Express, MongoDB

Roadmap

✅ Frontend completed
🔄 Backend development (user auth, product management, orders)
